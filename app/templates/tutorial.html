How To Make a Video

1. Save Script as Text File (.txt)

In a text editor (Google Docs, Microsoft Word), write out the script you'd like to use for your video narration. Insert image descriptions, in brackets, into the script where you would like it to appear. Be as specific as possible and double check your spelling, as the Video Creator selects a top search result from Google Images. Save the script as a text file (.txt)

In this example, the video will start with a picture of Romeo and Juliet art, then an image of William Shakespeare will appear after the narration says the phrase “written by”

[romeo and juliet art] Romeo and Juliet is a play written by [William Shakespeare] William Shakespeare. [Italy] It is set in Italy and is about the love [family feud romeo and juliet] between two young people from noble families that are enemies. It has been adapted to[romeo juliet opera] opera, [romeo juliet ballet] ballet, [romeo juliet tv show] television productions, and [romeo and juliet movie poster] movies.

Make sure there is an image description at the very start of the script, or else the program won't know what image to start the video with.

2. Upload Your Script

Click the Upload Script button to upload the text file from your computer to the Video Creator Tool.

3. Record Your Voice-over or Use a Computer-Generated Voice

A computer-generated voice-over of your script will automatically be used. However, if you prefer to use your own voice-over, create a recording of yourself reading the script. You should read the text exactly as written, without the image descriptions-we suggest copying and pasting the script and deleting the image descriptions so you have a clean copy to read from. Save your voice-over as an audio file (.wav or .mp3).

4. Upload Your Voice-over

Skip this step if you want to use a computer-generated voice for your video.

Select the check mark that says Use your own audio file. Click the Upload Audio File button to upload the audio file from your computer to the Video Creator Tool.

5. Use Your Own Images or Use Images from Google

By default, the program will search for your image descriptions on Google and use one of the top results as the image, but you can also upload your own images to use. Follow the corresponding instructions below depending on your choice.

6. Instructions for Using Your Own Images

  1. Check the Upload your own images option
  2. Confirm you have all the images you want to use downloaded. Make sure that all the images you want to use are the same size because if they aren't, the images will be stretched and squeezed.
  3. Click the Create Video button.
  4. You will be redirected to a page displaying your script. Each image description is in blue. Click each image description to upload an image to appear at that time. Make sure all image descriptions have an image uploaded for it.
  5. Once all image descriptions have an image uploaded, click the Create Video button.
  6. Wait to be redirected to your video

6. Instructions for Using Images From Google

  1. Select the usage rights you wish to use for the images. All the images used will have the selected usage rights.
  2. Click the Create Video button.
  3. Wait to be redirected to your video

7. Download Video

Once you see your video, click the Download Video button to download your video.
